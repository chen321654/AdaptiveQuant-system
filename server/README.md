## server项目结构

```shell
├── DataAnalysis
├── DataShow
├── History
├── Log
├── Message
├── RiskManagement
├── TradeDesicion
├── User
```

| 文件夹       | 说明                    | 描述                        |
| ------------ | ----------------------- | --------------------------- |
| `DataAnalysis`| 数据分析                | 数据分析功能 |
| `DataShow`   | 展示交易数据(股票、期货、基金)| 展示不同网站对的交易数据 |
| `History`   | 获取过去的交易数据         | 展示过去交易数据                |
| `Log`     | 日志                  | 运行日志 |
| `Message`    | 通知               | 基于交易数据改变发送通知 |
| `RiskManagement` | 风险管理 | 对风险进行预测和管理 |
| `TradeDesicion`      | 交易决策                  | 进行交易决策           |
| `User` | 用户功能 | 对用户进行管理 |
